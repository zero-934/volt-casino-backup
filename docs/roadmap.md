# Jett Casino Development Roadmap

This roadmap outlines the strategic phases for Jett Casino's development, growth, and market expansion, aligned with our pre-seed funding goals.

## Phase 1: Foundation & MVP (Complete - April 2026)
*   **Status**: COMPLETE
*   **Key Milestones**:
    *   Development and deployment of 11 unique, provably fair games.
    *   Implementation of `ProvablyFairRNG` (xoroshiro128+).
    *   Modular 3-file architecture (Logic/UI/Scene) for all games.
    *   Shared `SlotEngineLogic` and `SlotAnimator` for rapid slot development.
    *   Basic React-based lobby shell (`jett-landing`).
    *   Full Jest test coverage for core logic.
    *   GitHub Actions CI/CD for automated testing and GitHub Pages deployment.
    *   Doom Crash (world-first FPS crash game) launched.
    *   Surge & Inferno (novel cluster pays with cascade) launched.
    *   PIN-gated access to the game lobby.
    *   Deep-link routing for individual games.

## Phase 2: Platform Enhancement & Beta Launch (Q2 2026 - $500k Pre-Seed Funding)
*   **Status**: FUNDING REQUIRED
*   **Key Milestones**:
    *   **Casino Audio Pack Integration**: Professional sound effects and background music for all games, enhancing user immersion.
    *   **Analytics Implementation**: Integrate Supabase and PostHog for comprehensive user behavior tracking, game performance metrics, and responsible gambling monitoring.
    *   **Solana Devnet Wallet Integration**: Allow users to connect Solana wallets (e.g., Phantom) to interact with games on Devnet using test tokens.
    *   **Demo Mode & Free-to-Play**: Implement a robust demo mode allowing users to experience games without real funds.
    *   **Responsible Gambling Features**: Implement self-exclusion, deposit limits, and clear warning notices.
    *   **Mobile PWA Optimization**: Enhance performance and user experience for mobile progressive web app (PWA) deployment.
    *   **Enhanced Lobby & UX**: Significant UI/UX improvements for the game lobby, user accounts, and navigation.
    *   **Community Building**: Initiate Discord/Telegram channels, run Devnet-based contests.

## Phase 3: Mainnet Launch & Regulatory Compliance (Q3 2026)
*   **Status**: IN PROGRESS / DEPENDENT ON PHASE 2 COMPLETION
*   **Key Milestones**:
    *   **Solana Mainnet Integration**: Full integration with Solana mainnet for real-money wagering, JETT token transactions, and on-chain provable fairness.
    *   **JETT Token Launch & Staking**: Official launch of the JETT token, enabling staking for house edge share and governance.
    *   **Anjouan B2C Gaming License Acquisition**: Finalize and secure the B2C gaming license, enabling legal operation in target markets.
    *   **10 New Games**: Develop and launch 10 additional unique games, further expanding the "Interactive Risk Management Gaming" portfolio.
    *   **Multi-language Support**: Implement localization for key target markets (e.g., Hindi, Tagalog, Swahili).
    *   **KYC/AML Integration**: Implement necessary Know Your Customer (KYC) and Anti-Money Laundering (AML) checks in compliance with Anjouan regulations.

## Phase 4: Expansion & B2B Licensing (Q4 2026)
*   **Status**: PLANNING
*   **Key Milestones**:
    *   **B2B Licensing Program**: Launch a white-label program for other platforms to license Jett Casino games and the underlying technology.
    *   **Partnership Expansion**: Forge strategic partnerships with other Web3 projects, payment providers, and influencers.
    *   **Esports Integration**: Explore integrating skill-based games into competitive esports formats.
    *   **Series A Funding Preparation**: Begin preparing for a Series A funding round to support aggressive global expansion and platform innovation.
    *   **Decentralized Governance Rollout**: Further decentralize platform governance through the JETT DAO.
